## Description

[Nest](https://github.com/nestjs/nest) Boilerplate Backend Typescript.

## Project setup

```bash
pnpm install
```

## Compile and run the project

Development

```bash
pnpm run start
```

Watch mode

```bash
pnpm run start:dev
```

Production mode

```bash
pnpm run start:prod
```

## Run tests

Unit Test

```bash
pnpm run test
```

e2e tests

```bash
pnpm run test:e2e
```

Test coverage

```bash
pnpm run test:cov
```

## Nest Script

Generate Full Domain Module

```bash
nest g res <domain>
```

This will generate full modul entity, example entity user

```
└── 📁user
    └── 📁dto
        ├── create-user.dto.ts
        ├── update-user.dto.ts
    ├── user.controller.spec.ts
    ├── user.controller.ts
    ├── user.entity.ts
    ├── user.module.ts
    ├── user.service.spec.ts
    └── user.service.ts
```

## Script Custom

Command TypeORM

```bash
pnpm orm
```

Generate Migration from Entity

```bash
pnpm migrate:generate
```

Create New Empty Migrationn

```bash
pnpm migrate:create
```

Run Migration

```bash
pnpm migrate:run
```

Run Seeder

```bash
pnpm seed
```
